Если папка venv не существует, выполните следующие команды...

1. python3 -m venv venv

    MacOs/Linux
        2. source venv/bin/activate
    Windows
        2. ./venv\Scripts\activate.bat

3. pip3 install -r requirements.txt
4. python3 manage.py runserver
5. Happy hacking :)
